# Extract
Place this app in **nextcloud/apps/**

## Supported

* Zip 
* Rar
* Tar
* Gzip
* 7z
* Deb
* Bzip2

## Requirements

* Rar PHP extension ```pecl -v install rar``` or unrar ```sudo apt-get install unrar``` 
* p7zip ```sudo apt-get install p7zip p7zip-full```
